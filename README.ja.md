# vr-one2twenty

スピードと正確さが試される、テンポの速いVR/ARゲームです。ターゲットを順番通りにできるだけ早くタッチすることが目標です。

- **「One to Twenty」をプレイ (VR/AR):** [https://code4fukui.github.io/vr-one2twenty/](https://code4fukui.github.io/vr-one2twenty/)
- **「One to Seventeen SDGs」をプレイ (VR):** [https://code4fukui.github.io/vr-one2twenty/sdgs.html](https://code4fukui.github.io/vr-one2twenty/sdgs.html)

## 特徴

- **没入感のあるゲームプレイ:** WebXRを活用したフルVRおよびAR対応により、没入感のある体験を提供します。
- **2つのゲームモード:**
    - **One to Twenty:** 1から20までの番号が振られた球体を正しい順番でタッチします。
    - **SDGsバージョン:** 17の持続可能な開発目標（SDGs）のアイコンを順番にタッチしていく、教育的なバリエーションです。
- **ベストタイムの記録:** ブラウザのローカルストレージにベストタイムを保存し、自己ベスト更新に挑戦できます。
- **インタラクティブな操作:** VRコントローラー向けに設計されており、プレイヤーの動きやトリガー操作に反応します。
- **オーディオフィードバック:** ゲーム開始、ターゲットのタッチ、クリア時に効果音が鳴ります。

## 使用技術

- [Three.js](https://threejs.org/) - VR/AR体験のコア部分に使用。
- [A-Frame](https://aframe.io/) - SDGs VRバージョンに使用。

## クレジット

- 効果音は[効果音ラボ](https://soundeffect-lab.info/sound/button/)の素材を使用しています。
- SDGsアイコンは[Code for Sabae SDGs_jaプロジェクト](https://code4sabae.github.io/SDGs_ja/)のものを使用しています。
